# Test-Technique-Smart-X

## Installation

Clone the repository, then run the following commands:

```bash
touch .env
touch ./frontend/.env
```
Fill the .env file following the .env.example, then run :

```
sudo docker-compose up --build
```

## Usage

Frontend:

```bash
cd frontend/
npm install
expo start
```

Backend:

```bash
cd backend/
npm install
npm start
```
